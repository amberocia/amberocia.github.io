What Interactivity means on the Worldwide Web

If the term interactivity means the influence and working together of two things upon one another. What then does interactivity mean on the world wide web. Well loosely I would say that it is how the user communicates with and in the world wide web and vice versa. The web allows synergy between sender and receiver.

On the world wide web users have the ability to search for information, follow links, pull drop down menus, download videos, pictures etc. and by doing this they are afforded a more active role in consumption on the WWW. Users are able to interact with the worldwide web and through the worldwide web as a result of user interface (UI) a term often talked about in conjunction with user experience (UX), “which may include the aesthetic appearance of the device, response time and the content that is presented to the user within the context of the user interface. Both terms fall under the concept of human-computer interaction (HCI), which is the field of study focusing on the creation of computer technology and the interaction between humans and all forms of IT design. The user interface (UI) is the point of human-computer interaction and communication in a device.” (Churchville, 2022).

So then surely this means that just because you can go to a website etc. this constitutes as interactivity right even if there’s just text. Well honestly in my opinion yes it does (I have simple opinions), however it does not necessarily work like that. I would say the action of clicking the link which takes you to the website is interactive but if you get to the website itself and there’s just words but noting to actually do then surely its not interactive as mentioned before you’d need UI and UX for that. There should be buttons that take you to different pages, the ability for a user to reduce their screen size and still be able to access the website. The ability for the user too chooses which parts of that website they would like to engage with and certain design choices that attract the user to the website it should be easy to navigate through and not make the user feel lost. The combination of design and functionality is something every designer wishes to implement in a site.

Take Instagram’s website for example there are so many ways for the user to interact whether it be making posts, adding stories, commenting on posts, watching reels you name it, it gives the user a feel of freedom of choice. However, are those choices really all that free. Instagram and many other websites, apps are specifically designed to make the user act a certain way, to give the user a feeling/illusion of free choice and control.
